# Leaflet.Measurable

Leaflet measure control made with Leaflet.Editable.

## Dependencies

- leaflet > 1.0.0
- leaflet-editable

## Usage

```
map.addControl(new L.MeasureControl());
```

You can also add a position option for L.MeasureControl.

```
map.addControl(new L.MeasureControl({position:"topright"}));
```
